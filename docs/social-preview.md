# Social Preview Copy

GitHub social preview is configured in repository settings, not by README alone.
Use the existing `assets/hero.png` as visual direction, then upload a dedicated
1280 x 640 PNG/JPG preview in GitHub Settings.

## Recommended text

```text
Academic Paper Writing Skill
Evidence before prose
Literature Matrix • Claim Ledger • Integrity Check
No fake papers. No fake results.
```

## Visual direction

- large academic cap over a clean evidence matrix;
- three small labels: `Sources`, `Claims`, `Results`;
- restrained visual style that reads as academic workflow, not ghostwriting;
- enough empty space for text to remain legible in link previews.

## Avoid

- "AI writes your paper" language;
- promises of zero academic integrity risk;
- crowded paper piles or generic office imagery;
- screenshots that are unreadable at preview size.
