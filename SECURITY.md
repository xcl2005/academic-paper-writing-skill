# Security Policy

Academic Paper Writing Skill is designed for high-trust academic work, so privacy and integrity matter.

## Supported versions

The `main` branch is the active version.

## Reporting a vulnerability

Please open a GitHub issue if you find:

- instructions that encourage fabricated citations or fabricated results;
- hidden behavior in scripts;
- unsafe handling of private papers, drafts, advisor feedback, student records, or unpublished results;
- prompts that attempt to override no-fabrication or human-review rules;
- external-skill behavior that mutates core modules without explicit review.

Do not include private manuscripts, unpublished results, institutional documents, personal data, tokens, or credentials in the report.

## Security and integrity model

This skill cannot guarantee correctness. It enforces a workflow that makes claims, sources, assumptions, and evidence easier to inspect before final academic decisions.
