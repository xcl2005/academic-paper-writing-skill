# Integrity Checklist

- [ ] No fabricated citations.
- [ ] No fabricated results.
- [ ] SOTA/model/venue facts checked if current.
- [ ] Claims mapped to evidence.
- [ ] Strong claims qualified when evidence is weak.
- [ ] Figures/tables referenced and not misleading.
- [ ] Statistics/repeated runs checked when relevant.
- [ ] Reproducibility details recorded.
- [ ] School/venue requirements not invented.
- [ ] Human confirmation requested for high-stakes final submission.
