# Undergraduate Thesis Proposal Demo

This demo shows how to turn a graduation project idea into a proposal scope without inventing local school requirements.

## Input

```text
Use $academic-paper-writing-skill to turn my RAG-based academic assistant idea into an undergraduate thesis proposal scope and evidence checklist.
```

## First outputs

| File | Purpose |
|---|---|
| `requirement_discovery_log.md` | Records confirmed, missing, and assumed school/advisor requirements |
| `scope_ladder.md` | Separates MVP, standard target, and stretch goals |
| `graduation_evidence_map.csv` | Maps each thesis section to implementation, tests, screenshots, data, or official requirements |
| `assumption_register.md` | Keeps uncertain requirements visible until confirmed |

## Good stopping point

```text
Unknown: school template, defense rubric, advisor workload expectation.
Proceeding: draft an outline with compliance notes marked for human confirmation.
```
