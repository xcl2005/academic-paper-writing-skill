# Undergraduate Thesis Scope Ladder Sample

## Topic

RAG-based academic assistant with evidence tracking for thesis writing.

## MVP

- Ingest a small set of user-provided sources.
- Produce a literature matrix with verification status.
- Produce a claim ledger that blocks unsupported claims.
- Keep unknown school requirements marked as unknown.

## Standard Target

- Add novelty-check notes against verified prior work.
- Add an integrity checklist before prose generation.
- Generate a thesis outline only after the evidence map is inspectable.

## Stretch

- Add experiment matrix for retrieval quality and citation accuracy.
- Add reviewer-style critique of the thesis draft.
- Add export-ready tables for advisor review.

## Explicit Non-goals

- No fabricated citations.
- No invented advisor or school rules.
- No SOTA claims without verified evidence.
